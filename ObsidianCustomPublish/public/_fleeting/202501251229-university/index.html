📌 Mason Transfer & Degree Cheat Sheet #

🎯 Best Degree Options for You #

  1. 🔹 BS in Applied Computer Science (ACS)

    • Best if you want software development, web applications, and interactive media.
    • Relevant concentrations:
      • Software Engineering – Expands on your API & full-stack development.
      • Geoinformatics – If interested in data visualization & analytics.
      • Game Design – If you want to explore interactive applications.

  2. 🔹 BS in Information Technology (IT)

    • Best if you want a mix of web development, cloud computing, and system architecture.
    • Relevant concentration:
      • Web Development & Multimedia – Builds on your UI/UX & CMS experience.

  3. 🔹 BS in Computer Science (CS)

    • Best if you want to go deep into software engineering, AI, and system security.
    • Considerations:
      • Requires calculus, discrete math, and algorithms – check prerequisites.

  4. 🔹 BIS (Bachelor of Individualized Study) – Technology & Innovation

    • Best if you want flexibility to mix computer science, data analytics, and IT solutions.
    • Can incorporate transfer credits & professional experience.

✅ Key Steps to Take #

1️⃣ Transfer Credit & Work Experience Review #

  • 🔹 Contact Mason Admissions to evaluate your NOVA transcript.
  • 🔹 Check Prior Learning Credit (PLA) for professional experience:
    • Portfolio-based work credit may apply to web dev & IT courses.

2️⃣ Course & Prerequisite Review #

  • 🔹 If choosing CS or ACS, ensure you have:
    • 🔸 Math (Calculus, Discrete Structures)
    • 🔸 Programming (Java, Python, or C++)
      (Your experience with JavaScript & APIs helps!)
  • 🔹 If choosing IT or BIS, fewer math prerequisites required.

3️⃣ Prepare for Application & Enrollment #

  • 🔹 Request official transcripts from NOVA
    ( Order transcripts here)
  • 🔹 Contact Mason Admissions Office
    • 📧 Email: admissions@gmu.edu
    • 📞 Phone: 703-993-2400
  • 🔹 Review Mason’s transfer policies
    ( Mason Transfer Guide)

4️⃣ Plan Your Course Load #

  • First Semester Focus (if prerequisites needed):

    • 🔸 Discrete Math (if BS in CS/ACS)
    • 🔸 Intro to Programming (Java or Python)
    • 🔸 Database or Networking Fundamentals (for IT & web development focus)

  • Second Semester Focus:

    • 🔹 Advanced Web Systems (if ACS/IT)
    • 🔹 Data Structures & Algorithms (if CS)
    • 🔹 Hands-on Software Development (relevant to all paths)
